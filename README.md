MYSERVER - Página de Planos

Este projeto é uma landing page simples para apresentação de planos de hospedagem, com foco em design moderno e responsivo.

📸 Preview

A página exibe três planos principais:

Essencial
Premium
Deluxe

Com destaque visual para preços, benefícios e botão de ação.

🧩 Funcionalidades
Layout moderno e minimalista
Seção de planos com preços destacados
Navegação simples no topo:
Home
Planos
Suporte
Contato
Botões de chamada para ação ("Assine Já")
Design com cores escuras + destaque em verde neon
🛠️ Tecnologias Utilizadas
HTML5
CSS3
